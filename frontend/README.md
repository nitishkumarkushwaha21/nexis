# Frontend - Resume Screening UI

## Setup
1. Copy `.env.example` to `.env`
2. Keep dummy backend URL or update later
3. Run:

```bash
npm install
npm run dev
```

## Environment
- `VITE_API_BASE_URL=http://localhost:5000`

## Features
- Resume upload form (PDF/DOC/DOCX)
- Optional target role input
- Client-side file type + size validation
- Structured result sections for summary, skills, experience, education, projects, strengths, weaknesses, suggested roles, and role match
- Error + loading states
